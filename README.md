# Bean 이란 ?
- Java에서 인스턴스 생성
```
 Book book = new Book();
```
- Bean은 컨테이너가 관리하는 객체
  - 객체의 생명주기를 컨테이너가 관리
  - 객체를 싱글톤으로 만들것인지, 프로토타입으로 만들 것인지
----
Bean은 컨테이너에서 관리하게 해주는 것
